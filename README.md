# IDK

2D Graphics library written in C and OpenGL.

## Work in progress

This project is still in development and will most likely change a lot.

## Current Features
- Window
- Shape Rendering
- Texture Rendering
- Batch Rendering
- Keyboard/Mouse/Gamepad inputs
- Camera transformations
- Shader
- Tilemaps
